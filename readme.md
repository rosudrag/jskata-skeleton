# Javascript TDD Framework

Based on :
* karmajs
* webpack
* mocha
* specrunner

Utils:
* event dispatcher - message passsing pattern

## To run:

```karma start```

## To Write new test:
* add your test files in /test
* follow helloWorldTest.js example


## To Instal
* ```npm install```
* ```npm install -g karma-cli```
